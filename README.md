

# 如何使用
进入浏览器插件商店-> 下载 Tampermonkey -> 添加脚本 -> 代码复制 -> 保存 -> 访问youtube.com 即可

# 主要功能

- 展示字幕
- 总结

# 为什么开发？

我有一个快速看一系列youtube上AI教程的需求，但一个个看太费时间，我本以为AI发展到现在，这种应用应该很多。我见到过很多类似功能的插件了，但很遗憾，这些插件要么收费，要么有额度。总之不如意太多了。我自己有API，所以还是喜欢用自己的，质量有保证
